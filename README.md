## 🎨 ASCII Generator

Tool membuat seni ASCII dari teks maupun gambar dengan berbagai pilihan font dan warna.

## ✨ Fitur
- **Text-to-ASCII**: Ribuan font tersedia lewat engine `pyfiglet`.
- **Image-to-ASCII**: Ubah foto atau logo brand kamu menjadi karakter teks.
- **Color Support**: Tampilan terminal berwarna untuk estetika premium.
- **Auto Export**: Simpan hasil karya ke folder `output/` secara otomatis.

## 🛠 Instalasi
1. Pastikan Python 3 terinstall
2. Clone
   ```
   git clone https://github.com/123tool/ASCII-Generator.git
   cd ASCII-Generator
3. Instal library yang dibutuhkan:
   ```bash
   pip install -r requirements.txt
4. Jalankan :
   ```
   python main.py
